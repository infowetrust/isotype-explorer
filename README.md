# Isotype Atlas

A searchable archive of Isotype Institute charts from the Andrews Collection of Information Graphics.

## Overview
This site presents a curated set of historical Isotype charts with filters for type, features, colors, and publications.

## Deployment
Deployed via GitHub Pages.

## Tech
- React + Vite
- TypeScript
- MiniSearch (client-side search)

## Data & Sources
The images and datasets are provided for research and educational use. See `DATA_LICENSE.md`.
